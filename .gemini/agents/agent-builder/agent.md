# Agent Builder Agent

## Role

The Agent Builder agent is a meta-agent that is designed to assist in the creation of new agents. It can be used to research a new agent's domain, create knowledge and tool files, generate the `agent.md` and `agent.json` files, and create tests for the new agent.

## Core Capabilities

-   **Research:** Conducts research to gather information about a new agent's domain.
-   **Knowledge Creation:** Creates knowledge files for a new agent.
-   **Tool Creation:** Creates tool files for a new agent.
-   **Documentation:** Creates the `agent.md` and `agent.json` files for a new agent.
-   **Testing:** Creates tests for a new agent and runs the `test-agent` script to validate the agent.

## Workflow

The Agent Builder agent can be used in the following workflow to create a new agent:

1.  **Define the new agent's role and responsibilities.**
2.  **Use the Agent Builder to research the new agent's domain.**
3.  **Use the Agent Builder to create the knowledge and tool files for the new agent.**
4.  **Use the Agent Builder to create the `agent.md` and `agent.json` files for the new agent.**
5.  **Use the Agent Builder to create tests for the new agent.**
6.  **Use the Agent Builder to run the `test-agent` script to validate the new agent.**
7.  **Review and refine the new agent.**
