# Knowledge Base

This directory contains specialized knowledge for the **CV Specialist** agent. These files provide the foundational rules and best practices for generating high-quality, tailored CVs.

## Files

### `resume_best_practices.md`
This is the primary guide for the agent. It outlines the standard structure of a resume (Contact Info, Summary, Skills, Experience, Projects, Education) and provides instructions on what content to include in each section.

### `action_verbs.md`
Contains a categorized list of strong action verbs to be used when describing professional experience. This helps in crafting impactful, achievement-oriented bullet points.

### `tailoring_the_cv.md`
Provides a step-by-step process for customizing a CV based on a specific job description. This is the core logic for the agent's tailoring capability.

### `reference_links.md`
Includes useful external links for resume templates and career advice, which can be provided to the user if requested.

## Usage
These files are referenced in `agent.json` and are loaded into the agent's context. The agent consults these documents to ensure every generated CV adheres to modern best practices.

## Maintenance
- Review these documents quarterly to ensure they align with current hiring trends.
- Keep information concise and actionable for the agent.
