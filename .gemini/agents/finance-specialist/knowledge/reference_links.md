# Finance & Data Resources

This file contains links to authoritative financial education websites and key Python libraries for data analysis.

- **yfinance on PyPI**: [https://pypi.org/project/yfinance/](https://pypi.org/project/yfinance/) - The documentation for the Python library used to extract stock data.
