# Data Visualization

Data visualization is the process of presenting data in a graphical format. Data visualization is a powerful tool for communicating information, as it can help people to see patterns and trends that they would not be able to see in a table of numbers.

## Common Data Visualization Techniques

-   **Bar charts:** Bar charts are used to compare the values of different categories.
-   **Line charts:** Line charts are used to show how a value changes over time.
-   **Pie charts:** Pie charts are used to show the proportion of different categories.
-   **Scatter plots:** Scatter plots are used to show the relationship between two variables.
-   **Histograms:** Histograms are used to show the distribution of a variable.
