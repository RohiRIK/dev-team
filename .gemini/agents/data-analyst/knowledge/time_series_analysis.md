# Time Series Analysis

Time series analysis is a statistical method that is used to analyze time-series data. Time-series data is a sequence of data points that are collected over time.

## Common Time Series Analysis Techniques

-   **Moving average:** A moving average is a technique that is used to smooth out time-series data.
-   **Exponential smoothing:** Exponential smoothing is a technique that is used to make short-term forecasts.
-   **Autoregressive Integrated Moving Average (ARIMA):** ARIMA is a statistical model that is used to make long-term forecasts.
