# Tableau for Data Analysis

Tableau is a business intelligence tool that is used to create interactive visualizations and reports. Tableau is a powerful tool for data analysis, as it can be used to connect to a variety of data sources, to clean data, and to create beautiful and informative visualizations.

## Key Features of Tableau

-   **Connect to a variety of data sources:** Tableau can connect to a variety of data sources, including Excel, SQL Server, and Oracle.
-   **Clean and transform data:** Tableau can be used to clean and transform data.
-   **Create interactive visualizations:** Tableau can be used to create a variety of interactive visualizations, including bar charts, line charts, and pie charts.
-   **Share reports:** Tableau reports can be shared with others.
