# Feature Engineering

Feature engineering is the process of using domain knowledge to extract features from raw data. These features can then be used to improve the performance of machine learning models.

## Common Feature Engineering Techniques

-   **Imputation:** Imputing missing values.
-   **Binning:** Converting continuous variables into categorical variables.
-   **One-hot encoding:** Converting categorical variables into numerical variables.
-   **Feature scaling:** Scaling features to a common range.
-   **Feature selection:** Selecting the most important features.
