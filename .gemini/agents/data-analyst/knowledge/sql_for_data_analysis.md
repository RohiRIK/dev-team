# SQL for Data Analysis

SQL is a powerful language for working with data. It can be used to extract data from databases, to clean data, and to analyze data.

## Common SQL Commands for Data Analysis

-   **SELECT:** The SELECT command is used to select data from a database.
-   **FROM:** The FROM command is used to specify the table that you want to select data from.
-   **WHERE:** The WHERE command is used to filter the data.
-   **GROUP BY:** The GROUP BY command is used to group the data by a certain column.
-   **ORDER BY:** The ORDER BY command is used to sort the data.
-   **JOIN:** The JOIN command is used to join two or more tables together.
