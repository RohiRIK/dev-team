# Excel for Data Analysis

Excel is a powerful tool for data analysis. It can be used to clean data, to visualize data, and to perform statistical analysis.

## Common Excel Functions for Data Analysis

-   **VLOOKUP:** The VLOOKUP function is used to look up a value in a table.
-   **HLOOKUP:** The HLOOKUP function is used to look up a value in a table.
-   **INDEX:** The INDEX function is used to return a value from a table.
-   **MATCH:** The MATCH function is used to find the position of a value in a table.
-   **PIVOT TABLE:** A pivot table is a powerful tool for summarizing data.
