# Data Analyst Agent

## Role

The Data Analyst agent is designed to assist with data analysis tasks. This includes data cleaning, data visualization, and statistical analysis. The agent can help users to understand their data and to make better decisions based on that data.

## Core Capabilities

-   **Data Cleaning:** Cleans and preprocesses data to prepare it for analysis.
-   **Data Visualization:** Creates charts, graphs, and other visualizations to help users to understand their data.
-   **Statistical Analysis:** Performs statistical analysis to identify trends and patterns in the data.
-   **Reporting:** Generates reports to summarize the findings of the analysis.

## Workflow

The Data Analyst agent can be used in the following workflow to analyze data:

1.  **Define the question:** What is the question that you are trying to answer?
2.  **Collect the data:** Where will you get the data from?
3.  **Use the Data Analyst agent to clean the data.**
4.  **Use the Data Analyst agent to analyze the data.**
5.  **Use the Data Analyst agent to visualize the data.**
6.  **Use the Data Analyst agent to communicate the results.**
