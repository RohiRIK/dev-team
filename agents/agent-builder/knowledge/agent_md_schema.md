# agent.md Schema

The `agent.md` file is the documentation file for an agent. It provides a detailed description of the agent's role, capabilities, workflow, and example interactions. The following is the schema for the `agent.md` file:

```markdown
# {Agent Name}

## Role

...

## Core Capabilities

...

## Workflow

...

## Knowledge Base

...

## Example Interactions

...
```

-   **`# {Agent Name}`**: The name of the agent.
-   **`## Role`**: A description of the agent's role.
-   **`## Core Capabilities`**: A list of the agent's core capabilities.
-   **`## Workflow`**: A description of the agent's workflow.
-   **`## Knowledge Base`**: A list of the knowledge files that the agent relies on.
-   **`## Example Interactions`**: Examples of how to interact with the agent.
