# Agent-Specific Tools

This directory contains tool documentation specific to the Security Analyst agent.

## Available Tools

This agent does not currently have any specific tools assigned to it.

## Integration with Fabric
This agent can use the following fabric patterns:
- `summarize`: To summarize long security reports or articles.
- `analyze_claims`: To analyze claims made in security-related documents.

## Custom Commands
- `/analyze_alert`: To trigger an analysis of a security alert.
- `/hunt_threats`: To start a threat hunting process.
