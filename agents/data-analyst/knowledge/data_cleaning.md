# Data Cleaning

Data cleaning is the process of preparing data for analysis by removing or modifying data that is incorrect, incomplete, irrelevant, duplicated, or improperly formatted. Data cleaning is a critical step in the data analysis process, as it can have a significant impact on the quality of the results.

## Common Data Cleaning Tasks

-   **Handling missing values:** Missing values can be handled by removing the rows with missing values, or by imputing the missing values with a reasonable estimate.
-   **Handling outliers:** Outliers are data points that are significantly different from other data points. Outliers can be handled by removing them, or by transforming them to be more in line with the other data points.
-   **Handling duplicates:** Duplicate data points can be handled by removing them.
-   **Handling inconsistent data:** Inconsistent data can be handled by correcting the inconsistencies.
