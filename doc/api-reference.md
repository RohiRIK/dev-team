# API Reference

This document provides a detailed reference for the APIs of the different agents in the system.

## Agents

Below is a list of the available agents and their capabilities. To get more information about a specific agent, you can use the `/switch-agent <agent_name>` command and then use the `/help` command.

*   **backend-developer**: Helps with backend development tasks.
*   **barista-bot**: Your virtual barista.
*   **cloud-architect**: Helps with cloud architecture design.
*   **content-creator**: Creates a wide range of written materials.
*   **cv-specialist**: Helps you create a professional CV.
*   **data-analyst**: Helps with data analysis tasks.
*   **database-admin**: Helps with database administration tasks.
*   **devops-engineer**: Helps with DevOps tasks.
*   **finance-specialist**: Your virtual finance advisor.
*   **frontend-developer**: Helps with frontend development tasks.
*   **github-manager**: Manages and maintains GitHub repositories.
*   **homelab-manager**: Manages your homelab.
*   **ml-engineer**: Helps with machine learning engineering tasks.
*   **network-engineer**: Helps with network engineering tasks.
*   **product-manager**: Helps with product management tasks.
*   **qa-tester**: Helps with quality assurance testing.
*   **scrum-master**: Your virtual scrum master.
*   **security-analyst**: Helps with security analysis tasks.
*   **system-administrator**: Helps with system administration tasks.
*   **tech-news**: Your source for the latest tech news.
*   **ui-ux-designer**: Helps with UI/UX design tasks.
*   **workout-trainer**: Your virtual workout trainer.
